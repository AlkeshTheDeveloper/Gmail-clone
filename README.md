# Gmail-clone

## About Project
<hr>
 A Clone of Gmail desktop version to show features using HTML,CSS and  Js languages.

 ## features
 <hr>
1.Select all messages 
2. Profile can be Viewed
3. Mark as important
4. starred message
5. Hide or display right side bar
6. Create message Box
7. Google App menu

## Known bugs
